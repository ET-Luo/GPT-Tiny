# Cursor Rules: GPT-Tiny

You are operating inside the `GPT-Tiny` repository.
Follow the conventions below strictly.

## Environment activation (Miniforge)
- Before running any training/evaluation scripts, automatically run:
  `source $HOME/miniforge3/bin/activate`
- Then list environments with:
  `conda env list`
- Ask which environment to activate, and only then run:
  `conda activate <env_name>`
